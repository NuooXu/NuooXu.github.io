# 问题描述
Given an integer, write a function to determine if it is a power of three.

Example 1:  
Input: 27  
Output: true  

Example 2:  
Input: 0  
Output: false  
# 解决思路
笨方法强行解的，官方题解有其他的方法可以之后参考研究一下。
```java
class Solution {
    public boolean isPowerOfThree(int n) {
        if(n<1) return false;
        
        while(n % 3 == 0) n /= 3;
        return n== 1;
    }
}
```
